# Antigravity Intelligence Hub

**실시간 지식 자동화 모니터링 대시보드**

## 🚀 빠른 시작

### 1. 백엔드 API 서버 시작

```powershell
# API 디렉토리로 이동
cd api

# 의존성 설치 (최초 1회만)
pip install -r requirements.txt

# FastAPI 서버 시작
python main.py
```

서버가 시작되면 `http://localhost:8000`에서 API가 실행됩니다.

### 2. 프론트엔드 대시보드 열기

브라우저에서 다음 파일을 엽니다:
```
dashboard/index.html
```

또는 Python 간이 서버로 실행:
```powershell
cd dashboard
python -m http.server 3000
```

그 후 브라우저에서 `http://localhost:3000` 접속

## 📊 주요 기능

### ✨ 실시간 모니터링
- 수집된 뉴스 건수 실시간 표시
- NotebookLM 업로드 현황 추적
- 실행 중인 작업 상태 확인
- 최근 업데이트 시각 표시

### 🎮 원클릭 제어
- **뉴스 수집**: 보험 전문지 3사에서 최신 뉴스 자동 수집
- **NotebookLM 업로드**: 수집된 뉴스를 NotebookLM에 자동 업로드
- **블로그 발행**: 네이버 블로그에 콘텐츠 자동 발행
- **카페 발행**: 네이버 카페에 콘텐츠 자동 발행

### 📝 실시간 로그
- WebSocket 기반 실시간 로그 스트리밍
- 각 작업의 실행 과정 실시간 확인
- 오류 발생 시 즉시 알림
- 과거 로그 조회 기능

## 🎨 디자인 특징

- **다크 모드**: 눈의 피로를 줄이는 프리미엄 다크 테마
- **글래스모피즘**: 현대적인 반투명 유리 효과
- **실시간 애니메이션**: 부드러운 전환 효과와 마이크로 인터랙션
- **반응형 디자인**: 모든 화면 크기에 최적화

## 📡 API 엔드포인트

### GET `/api/scripts`
사용 가능한 모든 자동화 스크립트 목록 반환

### GET `/api/status`
현재 시스템 전체 상태 반환 (뉴스 데이터, 실행 중 작업, 최근 로그)

### POST `/api/execute/{task_id}`
특정 자동화 스크립트 실행

### GET `/api/logs/{task_id}`
특정 작업의 최근 로그 조회

### WebSocket `/ws`
실시간 로그 스트리밍

## 🔧 트러블슈팅

### API 서버에 연결할 수 없습니다
- 백엔드 API 서버가 실행 중인지 확인하세요
- `http://localhost:8000`에 접속하여 API 상태 확인

### WebSocket 연결 실패
- 방화벽에서 8000번 포트가 차단되지 않았는지 확인
- 브라우저 콘솔에서 오류 메시지 확인

### 스크립트 실행 실패
- 각 스크립트의 의존성이 설치되어 있는지 확인
- `execution/` 폴더의 스크립트 파일이 존재하는지 확인
- 로그를 확인하여 구체적인 오류 원인 파악
