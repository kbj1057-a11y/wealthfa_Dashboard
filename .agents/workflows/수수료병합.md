# 수수료 월별 병합 워크플로우

## 📁 폴더 구조
```
수수료분석/
  └── {YYYYMM}수수료/          ← 보험사별 원본 엑셀 파일 (13개)
        (사업부)수입수수료산출_{보험사명}_{YYYYMM}.xlsx
  └── {YYYYMM}수수료_생손보통합.xlsx  ← 최종 산출물
```

## ✅ 실행 조건 확인
- [ ] `수수료분석/{YYYYMM}수수료/` 폴더에 13개 파일 존재 확인
- [ ] 이전 달 통합 파일과 구분되는 연월(YYYYMM) 확인

## 📐 병합 규칙 (고정)

### 공통 컬럼 구조 (생명/손해 동일)
| 위치 | 컬럼 | 설명 |
|------|------|------|
| A ~ X (1~24) | 번호 ~ 보험료 | 모든 회사 공통 |
| Y (25) | 환산성적 / 수정보험료 | Y열 통일명 |
| Z (26) | 수수료계 | 합계, Z열 고정 |
| AA ~ | 세부 수수료 항목 | 각사 고유 컬럼 |

### Y열 명칭 통일 규칙
- **생명보험사** → `환산성적`
  - `환산성적` (농협생명, 미래에셋생명) → 그대로
  - `TP` (삼성생명) → `환산성적`
- **손해보험사** → `수정보험료`
  - `수정보험료` (AIG, DB, KB, 롯데, 삼성화재, 하나, 현대해상, 흥국) → 그대로
  - `신월정산수정P` (메리츠화재) → `수정보험료`
  - `환산실적` (한화손해보험) → `수정보험료`

### 회사 분류 기준
- **생명보험사**: 파일명에 `생명` 포함
- **손해보험사**: 그 외 전부

## 📊 기대 산출물
- 시트1: `생명보험사_{YYYYMM}` → 약 64컬럼
- 시트2: `손해보험사_{YYYYMM}` → 약 89컬럼
- 헤더 색상: 파랑(공통) / 밝은파랑(Y열) / 빨강(수수료계) / 초록(세부항목)
- 틀 고정: Z2 (Y열까지 + 헤더)

## ⚠️ 주의사항
- 새 보험사가 추가되면 파일명 패턴 확인 필요
- Y열 컬럼명이 바뀌면 스크립트 상단 `*_Y_ALIASES` 딕셔너리 업데이트
- 공통 컬럼(1~24)이 변경되면 `COMMON_COL_COUNT` 값 수정
